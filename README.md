# 🚀 Web Developer Emprendedor

Sitio web desarrollado por **Andy Ortiz**, enfocado en ofrecer servicios de desarrollo web moderno, diseño digital y soluciones tecnológicas a medida.

---

## 🌐 Descripción

Este proyecto es una página web profesional que presenta los servicios de un desarrollador emprendedor, con el objetivo de atraer clientes, mostrar proyectos y ofrecer soluciones digitales personalizadas.

Está diseñado para ser rápido, moderno y adaptable a diferentes dispositivos (responsive).

---

## 🎯 Objetivo del proyecto

- Mostrar servicios de desarrollo web
- Captar clientes potenciales
- Presentar soluciones digitales personalizadas
- Crear presencia profesional en internet
- Promocionar servicios en ciudades como Riobamba y otras

---

## 📂 Estructura del proyecto

El proyecto incluye los siguientes archivos principales:

- `index.html` → Página principal
- `apps-a-medida.html` → Servicios de desarrollo de aplicaciones personalizadas
- `desarrollo-web-riobamba.html` → Página enfocada en servicios locales
- `personalizador.html` → Herramienta o sección de personalización
- `temp_index.html` → Archivo temporal de pruebas o desarrollo
- `package.json` → Configuración del proyecto
- `package-lock.json` → Control de dependencias
- `vite.config.js` → Configuración del entorno de desarrollo con Vite

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Vite (para desarrollo rápido)
